# Real-Time Trend Detection on Social Media using Deep Learning

## Project Description
This project detects emerging trends and popular topics from social media posts in real time using deep learning techniques. It analyzes posts related to sports, cinema, education and technology, and predicts whether a topic is trending or not. An interactive dashboard is also included for live trend monitoring.

---

## Modules Implemented

### 1. Data Collection and Preprocessing
- Social media text dataset loading
- Data cleaning
- Tokenization and preprocessing

### 2. Exploratory Data Analysis
- Trend distribution visualization
- Pattern analysis

### 3. Feature Engineering
- Keyword and trend feature extraction
- Sentiment-based trend labeling

### 4. Model Selection
- LSTM Deep Learning model for trend detection

### 5. Model Training and Evaluation
- Training and testing
- Accuracy, Precision, Recall evaluation

### 6. Model Optimization
- Hyperparameter tuning
- Overfitting reduction using Dropout

### 7. Real-Time Implementation and Integration
- Dynamic trend prediction
- Interactive dashboard using Gradio/Streamlit
